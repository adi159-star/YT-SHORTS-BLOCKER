# YouTube Shorts Blocker

A simple Chrome extension that hides YouTube Shorts content on YouTube when the extension is enabled.

## Install

## Installation

1. Download this repository as ZIP.
2. Extract the ZIP file.
3. Open chrome://extensions/.
4. Enable Developer Mode.
5. Click Load Unpacked.
6. Select the extracted YT-Shorts-Blocker folder.
7. Open YouTube and refresh the page.

## Notes

- The extension hides elements that contain `/shorts/` links and sections labeled with `Shorts`.
- It works on YouTube pages where content loads dynamically.
